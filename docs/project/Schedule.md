# 我的 Schedule

## 简介

**我的 Schedule** 是一个管理待办事项的小程序。你可以在小程序上创建接下来一段时间需要完成的任务，并选择上传到云端，然后方便的在其它设备上查看和管理。

当开始一个任务时小程序会将任务按照每个阶段 25 分钟，划分为若干个阶段，每完成一个阶段会有 5 分钟的休息时间。休息结束后小程序会提醒你继续完成接下来的阶段，直到完成所有阶段。
